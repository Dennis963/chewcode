# opencode_remote

`opencode_remote` contains shared Dart models and bridge-client code used by ChewCode's Flutter mobile app. It is part of ChewCode, an unofficial mobile/bridge companion for OpenCode, and is not affiliated with or endorsed by OpenCode upstream.

This package is intended to mirror the ChewCode bridge contract used by the mobile app. The bridge itself is documented in the repository root README as `chewcode_bridge` for internal/npm-facing source naming and packaged for Debian as `chewcode-bridge`.
